The template language describes a context-free grammar, which the player controls. The syntax is shown beneath the edit box.

Template language syntax:
template: rule+ rule: nonterm-identifier optional-max-usage? "=" nonterm-descriptor nonterm-identifier: "@" identifier nonterm-descriptor: optional-prompt? "{" rule-list "}" optional-prompt: "[" prompt "]" / "[" placeholder "|" prompt "]" rule-list: rule "|" rule-list / rule rule: optional-hint? grammar-symbol+ optional-hint: text optional-max-usage? "=>" grammar-symbol: nonterm-identifier / nonterm-descriptor / text optional-max-usage: "[" integer "]" placeholder: text prompt: text
Here identifier is a standard C-like variable name and text is any text (in which quotes do not need to be escaped, but any of the characters "@#=<>\[]{}" must be escaped by preceding with a backslash character "\").
